Import prices from a CommerceML 1.0 upload.

== Description ==

Use the MCode CommerceML Import plugin to to get prices for goods downloaded from a CommerceML format file.

**Currency transfer.** If a different currency is used in the CommerceML file, indicate the rate and get prices in the desired currency

**Suitable for WooCommerce.** Automatically creates a Product ID field for products. All products will be synchronized in this field.

= Links =
* [Website](https://www.mcode.su)
* [GitHub](https://github.com/MCODEDOTSU/mcode-commerceml-import)

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Mcode CommerceML Import"
3. **Activate** Advanced Custom Fields from your Plugins page
4. **Click** on the new menu item "Import prices from a CommerceML 1.0 upload"
5. **Set** the exchange rate, the number of decimal places, select the import file and click "Import"
